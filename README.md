# Eye — Music Study

Lecteur PDF léger pour partitions (Electron + PDF.js), pensé pour l’étude : fit width/height, mode 2 pages, zoom fluide, rendu lazy (rapide même sur gros PDF).

## Features
- Rendu **lazy/virtualisé** (charge vite, render seulement ce qui est visible)
- **Fit Width / Fit Height**
- **1 page / 2 pages**
- **Zoom + / - / reset**
- **Ctrl + molette** pour zoomer
- Support fullscreen

## Raccourcis
- `C` : plein écran
- `B` : toggle fit (width/height)
- `P` : toggle 1 page / 2 pages
- `+` / `-` : zoom
- `0` : reset zoom
- `Ctrl + molette` : zoom
